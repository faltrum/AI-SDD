# CLAUDE.md

This repository uses the AI Spec-Driven Development (AI-SDD) framework.

The `.ai/` directory is the single source of truth for the engineering process.

When working in this repository:

1. Read the relevant documents from `.ai/` before making changes.
2. Follow the workflow defined in `.ai/workflow.md`.
3. Respect the principles defined in `.ai/manifesto.md`.
4. Use the templates in `.ai/templates/` when creating engineering artifacts.
5. Follow the project rules documented in `.ai/rules/`.
6. Do not invent engineering processes or duplicate documentation already defined in `.ai/`.

If requirements are incomplete or ambiguous, ask for clarification before proceeding.

Always align your work with the AI-SDD framework.
