# AI-SDD Agent Instructions

This repository uses the AI Spec-Driven Development (AI-SDD) framework.

The `.ai/` directory is the single source of truth for the engineering process.

Before starting any engineering task:

1. Read the relevant documents from `.ai/`.
2. Follow the workflow defined in `.ai/workflow.md`.
3. Respect the principles defined in `.ai/manifesto.md`.
4. Use the templates in `.ai/templates/` when creating engineering artifacts.
5. Do not invent processes or duplicate documentation already defined in `.ai/`.

If requirements are unclear, ask for clarification instead of making assumptions.

Always follow AI-SDD before generating code.
